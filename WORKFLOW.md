## 🔐 隐私保护设计

### JSON 文件包含什么？

```json
{
  "md5": "a1b2c3d4...",           // 文件哈希，不可逆
  "source_path": "D:/photos/...", // 本地路径，不上传
  "width": 1920,                  // 尺寸
  "height": 1080,                 // 尺寸
  "size_bytes": 524288,           // 大小
  "format": "JPEG",               // 格式
  "description": "一张夕阳下的海滩照片...", // 文字描述
  "tags": ["海滩", "夕阳", "风景"],        // 标签
  "compressed": false,            // 压缩标记
  "created_at": "2026-03-14..."   // 时间戳
}
```

### 隐私分析

| 字段 | 隐私风险 | 说明 |
|------|----------|------|
| MD5 | ✅ 无 | 哈希值，无法反推原图 |
| source_path | ✅ 无 | 本地路径，不上传云端 |
| 尺寸/大小/格式 | ✅ 无 | 元数据，不包含内容 |
| description | ✅ 低 | 文字描述，不包含人脸等 |
| tags | ✅ 无 | 通用标签 |

**结论**：JSON 文件本身**不包含可识别个人隐私的内容**，可以安全上传到云端。
